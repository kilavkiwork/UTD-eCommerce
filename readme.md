# Тестове завдання для компанії UTD

### [Сайт компанії](https://utdweb.team)

## Разработать/сверстать (HTML, CSS, JS) секцию "Best sellers".

Дизайн:
[desktop](https://www.figma.com/design/SldUom7WY4YDCM6E4Ae8xr/Shopify-Theme--2?node-id=140-9491&t=SCTfSwCGcWd2aoVl-0)
[mobile](https://www.figma.com/design/SldUom7WY4YDCM6E4Ae8xr/Shopify-Theme--2?node-id=458-70158&t=c59Rx7dUHshr3SZo-0)

1. Для написания стилей использовать "BEM".

2. В мобильном дизайне есть [3 кнопка](https://monosnap.ai/file/jaHuedM5vknbjU4IquMIBciKQaAKJF), ее делать не нужно.

3. Контент для двух других табов (картинка продукта, его название, цена - подобрать самостоятельно).

4. Картинка продукта, должна быть сверстана используя тег `<picture>`, чтобы для мобильного разрешения: `(max-width: 991px)` и `(max-width: 420px)`, выводить картинку меньше размером.

   - Так же, должны быть указаны размеры для `1x, 2x` плотности пикселей экрана.

5. У продукта могут выводится [теги](https://monosnap.ai/file/IxMmetuHtdsNaRY4s0LkzMIzYMTjl3).
   Все теги (если, например, их будет 3), должны иметь стили, как базовый тег `"New"`. Если `"tag" == "Sale"`, тогда он должен быть красным, как в дизайне (использовать модификатор).

6. У продукта ссылкой на продукт должны быть: картинка, название продукта.

7. При загрузке страницы, первый блок должен быть выбран по умолчанию. Всегда должен быть активным один блок (нельзя нажать на активный блок, чтобы он стал неактивным и контент в правой колонке скрылся).

8. При нажатии на имя [коллекции](https://monosnap.ai/file/p2yqmtAX8uzJ5TJ6szRJciWh3OA3U9), должен меняться контент в правой колонке. То есть, показываться другой продукт.

9. Кнопка с сердцем - не имеет никакого функционала. Просто сверстать.

10. Кнопка с глазом - при клике, должен открыться небольшой попап (модальное окно) шириной `~ 450px` с надписью: `"[PRODUCT NAME] has been added to the your cart."` Где `[PRODUCT NAME]` - имя текущего активного продукта.
    - При открытии попапа на странице должен появится overlay (полупрозрачная темная подложка), а поверх overlay должен находится по центру экрана попап.
    - Скрол на странице должен быть заблокирован.
    - В попапе так же должен быть `крестик (кнопка)`, при клике на который попап закроется.
    - Закрыть попап так же можно кликнув на `overlay`.
